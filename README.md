# cpp_algorithm

Competitive programming and algorithm study in C++.

## Environment

- Compiler: GCC 16 (`g++-16`)
- Standard: C++17

## Usage

```bash
# Create a contest directory (a.cpp - g.cpp)
make new CONTEST=abc400

# Compile
make build FILE=abc/abc400/a

# Compile & run
make run FILE=abc/abc400/a
```

## Structure

```
cpp_algorithm/
├── template.cpp
└── abc/
    └── abc400/
        ├── a.cpp
        └── ...
```
