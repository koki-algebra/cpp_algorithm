# アルゴリズム基礎

競技プログラミングに必要なアルゴリズムと考え方を体系的にまとめた教材です。

## 学習の進め方

1. 各章を順番に読む
2. コードは手元で写経して動かす
3. 章末の練習問題を解く（解けなければ解説を読んで再挑戦）
