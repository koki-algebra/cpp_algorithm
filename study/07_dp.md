# 第5章：動的計画法（DP）

最も重要かつ幅広いアルゴリズム。「状態」を定義して、小さな問題の答えから大きな問題の答えを作る。

---

## 5-1. DP の考え方

### 基本の発想

「大きな問題をより小さな同じ構造の問題に分解する」

**フィボナッチ数列**で体験する：

$$F(n) = F(n-1) + F(n-2)$$

再帰で素直に書くと $O(2^N)$ だが、計算済みの結果を使いまわすと $O(N)$。

```cpp
vector<long long> dp(n + 1);
dp[0] = 0;
dp[1] = 1;
for (int i = 2; i <= n; i++) {
    dp[i] = dp[i-1] + dp[i-2];
}
```

### DP 設計の手順

1. **状態を定義**する：`dp[i]` は何を表すか？
2. **遷移式**を立てる：`dp[i]` はどう計算するか？
3. **初期値**を設定する
4. **答え**がどこにあるかを確認する

## 5-2. 1次元 DP：コインの問題

**問題**：面値 `coins[]` のコインがある。合計金額 X を作るための最小枚数を求めよ。

```
dp[x] = 金額 x を作るための最小コイン枚数
```

**遷移式**：

$$dp[x] = \min_j \bigl( dp[x - \text{coins}[j]] + 1 \bigr)$$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    vector<int> coins = {1, 5, 10, 50};
    int X;
    cin >> X;

    const int INF = 1e9;
    vector<int> dp(X + 1, INF);
    dp[0] = 0;

    for (int x = 1; x <= X; x++) {
        for (int c : coins) {
            if (x - c >= 0 && dp[x - c] != INF) {
                dp[x] = min(dp[x], dp[x - c] + 1);
            }
        }
    }

    cout << dp[X] << endl;
}
```

## 5-3. ナップサック問題（0/1 knapsack）

**問題**：N 個の品物があり、それぞれ重さ `w[i]`、価値 `v[i]`。  
重さの合計が W 以下になるように選んで、価値の合計を最大化せよ。

```
dp[i][j] = i 番目まで考えたとき、重さの合計が j 以下の最大価値
```

**遷移式**：

$$
dp[i+1][j] = \max \begin{cases}
dp[i][j] & \text{（$i$ 番目を選ばない）} \\
dp[i][j - w_i] + v_i & \text{（$i$ 番目を選ぶ、$j \geq w_i$ のとき）}
\end{cases}
$$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, W;
    cin >> n >> W;
    vector<int> w(n), v(n);
    for (int i = 0; i < n; i++) cin >> w[i] >> v[i];

    // dp[j] = 重さ j 以下で選んだときの最大価値（1次元に圧縮）
    vector<long long> dp(W + 1, 0);

    for (int i = 0; i < n; i++) {
        for (int j = W; j >= w[i]; j--) { // 後ろから更新（0/1 ナップサック）
            dp[j] = max(dp[j], dp[j - w[i]] + v[i]);
        }
    }

    cout << dp[W] << endl;
}
```

**注意**：0/1 ナップサック（各品物を1個まで）は後ろから更新。  
無制限ナップサック（何個でも使える）は前から更新。

## 5-4. 部分和問題

**問題**：N 個の整数から、いくつか選んで合計を S にできるか？

```
dp[j] = 合計 j が作れるか（true/false）
```

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, S;
    cin >> n >> S;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];

    vector<bool> dp(S + 1, false);
    dp[0] = true;

    for (int i = 0; i < n; i++) {
        for (int j = S; j >= a[i]; j--) {
            if (dp[j - a[i]]) dp[j] = true;
        }
    }

    cout << (dp[S] ? "YES" : "NO") << endl;
}
```

## 5-5. LCS（最長共通部分列）

**問題**：文字列 S と T の最長共通部分列（LCS）の長さを求めよ。

```
dp[i][j] = S の最初の i 文字と T の最初の j 文字の LCS の長さ
```

**遷移式**：

$$
dp[i][j] = \begin{cases}
dp[i-1][j-1] + 1 & (S[i-1] = T[j-1]) \\
\max(dp[i-1][j],\ dp[i][j-1]) & (\text{otherwise})
\end{cases}
$$

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s, t;
    cin >> s >> t;
    int n = s.size(), m = t.size();

    vector<vector<int>> dp(n + 1, vector<int>(m + 1, 0));

    for (int i = 1; i <= n; i++) {
        for (int j = 1; j <= m; j++) {
            if (s[i-1] == t[j-1]) {
                dp[i][j] = dp[i-1][j-1] + 1;
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
            }
        }
    }

    cout << dp[n][m] << endl;
}
```

## 5-6. DP 典型パターン一覧

| パターン | dp の定義例 | 代表問題 |
| --- | --- | --- |
| 1D DP | `dp[i]` = i 番目までの最適値 | コイン問題 |
| ナップサック | `dp[j]` = 重さ j 以下の最大価値 | 0/1 ナップサック |
| 部分和 | `dp[j]` = 合計 j が作れるか | 部分和問題 |
| 区間 DP | `dp[l][r]` = 区間 [l,r] の最適値 | 行列積の最適化 |
| グリッド DP | `dp[i][j]` = (i,j) までの最大/最小 | 経路の最大和 |
| 桁 DP | `dp[桁][上限フラグ]` | N 以下で条件を満たす数の個数 |

## 5-7. グリッド DP の例

**問題**：H×W のグリッドを左上から右下まで移動する。各マスに価値があり、価値の合計を最大化せよ。右か下にしか移動できない。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int H, W;
    cin >> H >> W;
    vector<vector<int>> a(H, vector<int>(W));
    for (int i = 0; i < H; i++)
        for (int j = 0; j < W; j++) cin >> a[i][j];

    const long long NEG_INF = -1e18;
    vector<vector<long long>> dp(H, vector<long long>(W, NEG_INF));
    dp[0][0] = a[0][0];

    for (int i = 0; i < H; i++) {
        for (int j = 0; j < W; j++) {
            if (i == 0 && j == 0) continue;
            long long best = NEG_INF;
            if (i > 0 && dp[i-1][j] != NEG_INF) best = max(best, dp[i-1][j]);
            if (j > 0 && dp[i][j-1] != NEG_INF) best = max(best, dp[i][j-1]);
            if (best != NEG_INF) dp[i][j] = best + a[i][j];
        }
    }

    cout << dp[H-1][W-1] << endl;
}
```
