## Repository Purpose

This repository is used for competitive programming and C++ questions.

## Code Style

When explaining with code, always base the code on `template.cpp`.
Anything already defined in `template.cpp` (includes, using declarations, #define macros, constants, etc.) should be omitted from explanation code — assume it's already in place and do not repeat it.

## Editing Policy

Do not edit any code until explicitly instructed to do so.
